# Roadmap

{% stepper %}
{% step %}
## Decentralized Arbitration

Move from a controlled arbiter allowlist to fully decentralized resolution through token-weighted voting. Anyone holding the governance token can participate in dispute resolution.
{% endstep %}

{% step %}
## Automatic Payouts

When a claim is settled, downstream actions fire automatically — insurance pays out, bridges release funds, prediction markets resolve. No manual intervention.
{% endstep %}

{% step %}
## Custom Resolution Rules

Every protocol has different requirements. Clarity Optimistic Oracle will support pluggable resolution strategies — faster windows, stricter thresholds, or entirely custom logic per use case.
{% endstep %}

{% step %}
## Protocol Fees

A small fee on each assertion creates a sustainable revenue stream that scales with adoption across the ecosystem.
{% endstep %}

{% step %}
## Ecosystem Standard

The long-term goal: be the default truth layer for every bridge, prediction market, insurance protocol, and DeFi app on Stacks.
{% endstep %}
{% endstepper %}
